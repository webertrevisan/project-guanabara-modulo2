Modelo de Caixas

Aninhamento de caixas

  • altura = hight
  • largura = width
  • Border = linha que circunda o conteudo
  • padding = acolchoamento ou preenchimento faz com que a borda fique distante do conteudo
  • margin = do border para fora
  • outline = Contorno ou traçado localizado fora do border

    • Tipos de Caixas

  • Box-level = ocupa toda a linha e espaço do box quebrando linhas (DIV é um dos box level mais famosos)
  • Inline-level = não quebra linha nao ocupa todo espaço da linha só o tamanho preciso(span é uma tag famosa no inline level)

    • Exemplos de Box-level e inline-level
